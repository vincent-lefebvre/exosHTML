Question 1
La déclaration DOCTYPE est-elle obligatoire ? Où doit-elle être placée ? Quelles informations contient-elle ?
Réponse 1 : Oui elle est plus que recommandée, donc obligatoire, car elle définit la nature du fichier, et elle se place en tête du code devant la balise <html>, elle contient le nom de l'élément racine.
Question 2
Que faut-il faire pour que le code HTML soit valide ?
Réponse 2 : Il faut respecter l'inclusion ou l'imbrication des éléments et les attributs propres à chacun d'entre eux puis faire valider le code par le W3C (World Wide Web Consortium)
Question 3
À quel emplacement est defini le jeu de caractères utilisé dans le document ?
Réponse 3 : le jeu de caractère est précisé dans un élément <meta> avec l'attribut charset inclus dans <meta charset="UTF_8">
Question 4
Où est déclaré le nom de l'élément racine du document ? Quel est son rôle et son nom ?
Réponse 4 : le nom de l'élément racine est donné dans DOCTYPE il s'agit de html ; c'est l'élément qui inclus tous les autres donc il est au sommet de la hiérarchie
Question 5
Quels sont les éléments enfants de l'élément HTML ?
Réponse 5 : les enfants (de premier niveau) sont <head> et <body>
Question 6
À quoi sert l'élement <head> ? À quoi sert-il ?
Réponse 6 : L'élément head est l'en-tête du document, il n'est pas visible dans le navigateur (sauf le title) : il contient BASE, LINK, META, SCRIPT, STYLE, TITLE. Seuls les éléments BASE et TITLE ne doivent être utilisés qu'une seule fois

Question 10
Comment déclarer les mots-clefs associés au site ? Quelle est l'utilité de cette déclaration ? Écrivez-un exemple
Réponse 10 : En insérant les mots-clefs dans la balise <meta name="keywords" content="Exercices eBook, HTML">
Question 11
Quel est le rôle de l'élément <base>
Réponse 11 : L'élément <base> définit l'URL de base à utiliser pour recomposer toutes les URL relatives contenues dans un document. Il ne peut y avoir qu'un seul élément <base> au sein d'un document.
Question 12
Peut-on écrire le code suivant dans la page ?
<body>
Bienvenue dans notre site <h1>Le site du jour</h1>
</body>
Réponse 12 : Ce code n'est pas conforme car l'élément BODY contient du texte directement, alors qu'il devrait etre inclut dans un élément enfant de BODY
