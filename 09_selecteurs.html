Aide mémoire sélecteurs CSS 3 et leurs propriétés

Les selecteurs CSS

  • * : sélectionne TOUS les éléments de la page HTML
  • .ma-jolie-class : sélectionne tous les éléments
  • #mon-unique-id : sélectionne tous les éléments ayant le nom d'ID
  • div, aside, article : sélectionne toutes les balises <div>, <aside>, <article>
  • nav a
  • nav a:hover

Propriétés CSS utilisés dans cette page

Décrivez leur usage et leurs valeurs possibles

  1. Propriétés pour le texte
    • font-family : famille de typo dans un ordre de priorité
    • font-size : taille des caractères ou de la typo
    • text-decoration : pour "décorer" le texte en ajoutant une ligne en dessous
    • text-align : alignement du texte, à gauche, à droite, ou au centre
  2. Propriétés pour les éléments
    • background : arrière-plan d'un élément
    • height : hauteur de l'élément
    • width : largeur de l'élément : tous les deux par rapport à la taille de la fenêtre si on indique une valeur proportionnelle en % ou en em
    • color : couleur de la typo, de la font ou du texte
    • left : position à partir de la gauche de la fenêtre du navigateur
    • line-height : interlignage, espace entre les lignes
    • float : permet de sortir un élément du flux / effet "flottant"
    • margin : marge extérieur d'un élément
    • padding : marge intérieur d'un élément
    • position : position de l'élément dans la page
    • top : position par rapport au haut de la fenêtre
    • transition : temps de transition lors d'une animation
